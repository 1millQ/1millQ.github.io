Install Bebbo GCC

1) Open terminal (if not already opened)

2) sudo apt install make wget git gcc g++ lhasa libgmp-dev libmpfr-dev libmpc-dev flex bison gettext texinfo ncurses-dev autoconf rsync «copy

3) git clone https://github.com/bebbo/amiga-gcc «copy

4) cd amiga-gcc «copy

5) make update «copy

6) sudo mkdir /opt/amiga «copy

7) sudo chgrp users /opt/amiga «copy

8) sudo chmod 775 /opt/amiga «copy

9) sudo usermod -a -G users ChangeToYourUserName «copy

10) make clean «copy

11) make clean-prefix «copy

12) make all NDK=3.2 «copy

13) make all-sdk «copy

14) cd ..

Emu68-tools


1) git clone https://github.com/michalsc/Emu68-tools «copy

2) cd Emu68-tools «copy

3) mkdir build install «copy

4) add text in top of file Emu68-tools/cmake/toolchain.make SET(TOOLCHAIN_PATH /opt/amiga) «copy

5) cd build «copy

6) cmake .. -DCMAKE_TOOLCHAIN_FILE=../cmake/toolchain.cmake «copy

7) make «copy
